Cara kerja aplikasi blokir iklan ini cukup sederhana. Yakni dengan me-resolve semua nama host (domain) sever iklan ke alamat IP localhost (127.0.0.1) atau IP non-routeable (0.0.0.0) sehingga tidak akan mendapatkan tanggapan semestinya. Untuk melakukan hal ini, biasanya aplikasi membutuhkan hak akses root agar bisa merubah file host. Sebagai contohnya adalah aplikasi paling populer AdAway yang juga saya pakai diawal-awal menggunakan ponsel Android.

Seiring berjalannya waktu, saya membutuhkan ponsel yang bisa digunakan untuk meng-akses aplikasi perbankan. Sehingga saya pun tidak bisa lagi secara leluasa melakukan rooting (sebelum munculnya Magisk) dan berusaha mencari solusi lain untuk memblokir tampilan iklan di website tertentu yang saya anggap terlalu intrusif.

Dalam proses pencarian ini, saya pun menemukan aplikasi-aplikasi VPN yang yang mampu menutup akses server iklan dengan file host yang sudah disediakan di sever aplikasi tersebut.

Singkat cerita, saya pun menemukan aplikasi DNS66 yang cara kerjanya sama seperti aplikasi VPN. Sedikit berbeda dengan Aplikasi VPN secara umum, DNS 66 beroperasi dengan membuat sever virtual yang dijalankan pada localhost perangkat kita. Sehingga saya pun bisa menentukan sendiri alamat server iklan mana saja yang akan saya blokir. Untuk lebih jelasnya bisa mencari informasi pada laman resmi pengembang aplikasi DNS 66 di sini.

Cara Memakai DNS 66 untuk Memblokir Iklan

Sebelumnya aplikasi ini pernah muncul di Playstore tetapi kemudian hilang. Jika kamu menemukan aplikasi ini di Playstore, aplikasi tersebut adalah aplikasi kloning, ada juga aplikasi sejenis bernama DNS Changer, namun malah menampilkan iklan ğŸ™„ dalam aplikasi. Untuk mendapatkan aplikasi ini, kamu mesti mengunduh file ini dari repo F-Droid atau rilisan di GitHub. Keduanya sama saja.

1. Download Aplikasi DNS 66 : Cukup, sekarang silahkan download aplikasi DNS 66 dari tautan ini dan pilih yang terbaru. Jika kamu bingung mesti mengunduh yang mana, pakai saja tautan ini untuk mengunduh DNS66 v0.6.0. Yakni rilis terbaru (ke-21) saat saya menulis artikel ini.

2. Pasang Aplikasi :Pasang APK yang kamu unduh. Biasanya akan muncul peringatan yang mencegah aplikasi dipasang. Kamu bisa mengatur agar diperbolehkan memasang aplikasi selain dari playstore di pengaturan ponsel. Kemudian buka aplikasi DNS66.

3. Memilih sumber file host : Sebelum memulai, pilih domain filters yang terletak di tab HOST. Dari pilihan ini kamu harus memilih satu dari penyedia sumber daftar server iklan (file host) yang tersedia. Sebagai langkah awal, pilih saja Adaway host file. Caranya, cukup menekan logo perisai yang masih putih (ada di kiri pilihan) hingga berubah menjadi perisai bermotif kotak-kotak. Lanjutkan dengan menekan ikon refresh yang ada pada pojik kanan atas untuk memperbarui file host. Tunggu saja hingga proses ini selesai, kemudian pilih tab START untuk kembali ke layar utama aplikasi.

Setelah VPN aktif, ditandai dengan munculnya tanda kunci pada notification bar kamu tidak akan lagi melihat iklan yang muncul. Baik itu di browser maupun aplikasi Android yang terpasang pada ponsel android kamu.

Kelebihan dan Kekurangan

Tidak seperti aplikasi sejenis, DNS 66 bekerja pada tingkatan DNS, yang berarti hanya lalu lintas DNS saja yang dialihkan melalui DNS66. Jika tidak tidak di temukan pada daftar, barulah dilempar ke DNS Resolver yang ditentukan pada tab DNS. Cara kerja ini (seperti yang sudah saya jelaskan di awal dan juga pada artikel sebelumnya tentang cara baru memblokir iklan) sangat berbeda jika dibandingkan dengan metode yang banyak digunakan oleh aplikasi-aplikasi serupa, yaitu dengan menyaring semua data lalu lintas yang ada untuk iklan tanpa melakukan proses filtering.

Untuk penggunaan sumber daya dan baterai ponsel, DNS66 juga terhitung ramah. Sebagai perbandingan, pada gambar di bawah ini, aplikasi AdGuard berada pada peringkat paling atas sebagai aplikasi yang paling banyak menguras baterai, sedangkan aplikasi DNS66 bahkan tidak muncul di dalam halaman statistik penggunaan baterai.

Cara kerja ini sangat efektif dan efisien. Terkait daftar server yang diunduh tersebut merupakan file yang sudah jadi. Yang karena bukan buatan lokal, tentu tidak punya daftar host iklan lokal (Indonesia). Sedangkan saya juga perlu menambahkan beberapa host iklan lokal tersebut. Namun bukannya tidak bisa dikustomasi layaknya aplikasi AdAway yang saya sebut diawal. Saya masih bisa (dan mau untuk ribet) menambahkan sendiri daftar host yang ingin di blokir.

Secara keseluruhan, aplikasi DNS66 (dan juga AdGuard) adalah aplikasi yang praktis. Terutama bagi pengguna yang tidak seperti saya, yang ribet beut. Dan yang terpenting adalah: Aplikasi DNS 66 tidak membutuhkan akses root karena tidak butuh melakukan perubahan file host yang berada pada partisi sistem. Sehingga bagi pengguna ponsel android yang masih membutuhkan beberapa aplikasi perbankan bisa dengan nyaman memakai DNS 66 untuk memblokir iklan.
